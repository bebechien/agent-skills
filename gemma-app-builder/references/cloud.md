# Cloud Deployment

This module covers deploying Gemma to the cloud for scalable, production-ready applications.

## Option

### Vertex AI
Google Cloud's fully managed AI platform.

**Setup:**
-   **Model Garden**: Search for "Gemma" in the Vertex AI Model Garden.
-   **Deploy**: Click "Deploy" to create an endpoint.
-   **Machine Type**: Select a GPU-enabled machine (e.g., L4, A100).
-   **API**: Access the model via the Vertex AI Prediction API.

**Example Prompt:**
"Walk me through deploying Gemma 7B to a Vertex AI endpoint."
